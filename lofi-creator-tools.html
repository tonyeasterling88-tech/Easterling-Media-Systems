LoFi Creator Tools

Desktop systems for long-form LoFi production.

This build family includes LoFi Forge for assembling long-duration videos and LoFi Stamp for branding, SEO metadata, and YouTube publishing workflows.

Back to BuildsView Content Hub

Project Facts

Apps
LoFi Forge and LoFi Stamp
Stack
Electron, React, TypeScript, FFmpeg
Output
Long-form MP4 LoFi videos
Workflow
Assemble, export, brand, publish

The Problem: long-form LoFi output has many repeatable production steps.

Video assembly

LoFi Forge creates long-form videos from still images or short video scenes, music, optional ambience, and export settings.

Branding pass

LoFi Stamp brands finished videos with PNG watermark overlays and optional CTA text through FFmpeg.

Publishing flow

LoFi Stamp docs define a planned YouTube OAuth upload flow with metadata, thumbnails, and Studio links.

The Current Build

LoFi Forge

Electron, React, Vite, TypeScript, FFmpeg CLI, `.lofi.json` project files, queue exports, scene motion, audio mixing, and Windows packaging.

LoFi Stamp

Electron desktop MVP with bundled FFmpeg support, watermark/CTA overlays, H.264 export, logs, presets, OpenAI SEO, and Google OAuth hooks.

Known gap

LoFi Forge memory notes say the editable renderer source is missing in that snapshot, while main/export/queue code remains workable.

Development Timeline

Assemble

Create/open projects, import scenes, configure motion and duration, add music and ambience.

Export

Run staged FFmpeg exports: preflight, render scenes, build audio, mix, mux, verify output.

Brand

Apply watermark and CTA overlays with LoFi Stamp, using deterministic presets where possible.

Publish

Move toward YouTube OAuth upload with title, description, tags, privacy, thumbnail, and success links.
