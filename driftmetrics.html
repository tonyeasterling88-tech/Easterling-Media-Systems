DriftMetrics

Experiment: TikTok metadata review dashboard.

DriftMetrics helps creators review TikTok video metadata and identify mood, timing, and content patterns. This build stays narrow on purpose: policy-complete pages, a protected review dashboard, and TikTok Login Kit / Display API OAuth scaffolding for app review.

Review scopes

Current focus

A clean review story: connect your own account, import metadata, and keep everything constrained to approved access.

The Problem: creators need pattern clarity, not more dashboards.

Review-first scope

The current version is scoped for TikTok app review: minimal data scopes, explicit behavior limits, and clear policy pages.

Metadata only

DriftMetrics focuses on the creator's own public video metadata to help surface timing and content patterns.

No automation

This build does not post to TikTok, automate engagement, scrape content, or request scopes outside profile + video list access.

The Current Build

Policy-complete surface

A public homepage plus Privacy, Terms, and Data Deletion pages keep the review footprint complete.

Protected review dashboard

A review dashboard shows connection state, demo-ready content, and what's intentionally unavailable outside approved TikTok access.

TikTok OAuth scaffolding

Login Kit + Display API OAuth start/callback routes support connecting a creator account and pulling basic profile + video list data.

Development Timeline

Review packet + policies

Ship a review-ready site with clear, consistent app review language.

OAuth + data import

Connect TikTok and import basic profile + video metadata using narrow scopes.

Creator insights loop

Generate pattern insights without expanding into posting, engagement automation, or scraping.

Broader rollout

Only expand scope after review approval and updated policy + review language.
